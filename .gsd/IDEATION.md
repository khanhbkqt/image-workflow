# Image Workflow — Ideation & Research

## Vision

A **macOS desktop app** for a visual, composable image generation studio — combining the best ideas from Google Labs' ImageFX, Whisk, and Flow into a node-based canvas where **ingredients** (reusable visual building blocks) are wired into **workflows** that produce images at scale with brand consistency.

This is a **private, personal tool** — not a SaaS product.

---

## Target User Persona

**Solo content creator / marketer (you)**
- Produces branded images regularly (social media, ads, blog headers, product shots)
- Needs visual consistency across campaigns
- Prefers visual, drag-and-drop interfaces over raw text prompts
- Values speed: batch generation > one-at-a-time iteration

---

## Core Problem

Current AI image tools are **linear and one-shot**: type a prompt → get an image → tweak → repeat. This fails because:
1. **Slow** — generating images one-by-one doesn't scale
2. **Inconsistent** — hard to maintain brand identity across many images
3. **Opaque** — no way to decompose, save, and reuse the "recipe" behind a good image

---

## Key Concepts

### Projects
A project is a **workspace/folder** that groups related canvases, ingredients, and generated images. Examples: "Summer Campaign", "Product Launch". You switch between projects like folders.

### Ingredients
Reusable building blocks — **created manually OR generated by AI**. Inspired by Whisk's subject/scene/style inputs.

| Type | Description | Example |
|------|-------------|---------|
| **Subject** | Main object/person/thing | "Golden retriever puppy", uploaded product photo |
| **Scene** | Background, environment | "Tropical beach at sunset", reference photo |
| **Style** | Aesthetic, mood, treatment | "Minimalist flat illustration", "cinematic lighting" |
| **Text Overlay** | Typography, captions | "Summer Sale 2026" |
| **Modifier** | Adjustments | "Warm tones", "16:9", "high contrast" |
| **Brand Kit** | Bundled style + modifiers | Company palette + typography + mood |

**Key insight**: Ingredients are AI-generatable. A "Generate Ingredient" node creates ingredients on-the-fly.

### Workflows
A **directed graph of nodes** on an infinite canvas. Nodes = ingredients, generators, transformers, outputs. Wires = data flow.

### Recipes
Saved workflows — reusable templates that can be cloned and parameterized.

## Detailed Use Cases

### UC1: Quick Single Image
> Drag Subject + Style → Connect to Generate → Get 4 variations → Download.

### UC2: Brand-Consistent Batch
> Load Brand Kit + list of Subjects → Batch Generator → 40 images, all on-brand.

### UC3: Style Exploration & Fan-Out
> One Subject → branch to 5 Styles → Generate all → compare side-by-side on canvas.

### UC4: AI-Generated Ingredients (Recursive)
> No background photo? "Generate Ingredient" node → describe scene → get Scene ingredient → wire into workflow.

### UC5: Ingredient Library Management
> Browse, create, tag, organize ingredients. Search & filter. Drag onto any canvas.

### UC6: A/B Testing Visual Variants
> Same Subject + Scene → branch into two Styles → generate both → compare → pick winner.

### UC7: Project Management
> Create projects as workspaces. Each project holds its own canvases, ingredient instances, generated images. Switch between projects from a home/dashboard screen.

## Node Types (Canvas Primitives)

| Node | Icon | Purpose |
|------|------|---------|
| **Ingredient** | 🧩 | Holds a reusable building block |
| **Generate** | ⚡ | Ingredients → AI backend → image(s) |
| **Generate Ingredient** | 🔮 | Creates a new ingredient from text/image |
| **Batch Generator** | 📦 | List of inputs → all combinations |
| **Style Fan-Out** | 🌿 | Splits input across multiple styles |
| **Compose** | 🔗 | Merges ingredients into a combined prompt |
| **Preview** | 👁️ | Displays image inline on canvas |
| **Output** | 💾 | Marks final output for download/export |
| **Brand Kit** | 🏷️ | Bundles style + modifiers |

## Key User Flows

### Flow A: First-Time (Onboarding)
```
Open App → Empty Canvas + Welcome Tooltip
  → Drag Subject → Drag Style → Wire to Generate
  → Generate → First image appears 🎉
  → Save as Recipe
```

### Flow B: Daily Power User
```
Open App → Recent Projects
  → Open "Summer Campaign" → Load saved workflow
  → Swap Subject → Adjust modifier → Generate All
  → Review → Approve → Export ZIP
```

### Flow C: Building a Recipe
```
New Canvas → Drag Brand Kit from Library
  → Generate Ingredient: "minimalist desk setup" → Scene created
  → Wire: Brand Kit + Scene + Product Photo → Compose → Generate
  → Review → Tweak → Save as Recipe: "Product Lifestyle Shot"
```

## Google Labs API Mapping

| Our Concept | Google Tool | Capability |
|-------------|-------------|------------|
| Text-to-image | **ImageFX** | Imagen 3/4, expressive chips |
| Subject/Scene/Style inputs | **Whisk** | Image-based inputs → Gemini captioning → Imagen |
| Unified workspace | **Flow** | Project-based studio, refinement, style transfer |
| Ingredient generation | **Whisk + ImageFX** | Generate reference images as workflow inputs |

**Backend:** Reverse-engineer Google Labs APIs via browser cookies.

## Technical Decisions

## Non-Goals (v1)

- ❌ Video generation
- ❌ Real-time collaboration
- ❌ Web app (desktop only)
- ❌ Custom model training / fine-tuning
- ❌ Image editing (cropping, retouching)
- ❌ Self-hosted AI models
- ❌ Cross-platform (macOS only for now)
