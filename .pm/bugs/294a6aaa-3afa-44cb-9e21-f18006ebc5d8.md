# Node connector handles not displaying

**Priority**: high
**Status**: open
**Reported by**: 496966e8-7138-4392-a374-0d6ca1ef5113

## Description

The input/output connector handles (circular dots on left/right edges of nodes) are not visible on canvas nodes (IngredientNode, PromptNode). This prevents users from wiring nodes together to build generation pipelines. Root cause: likely a CSS stacking context conflict caused by backdrop-filter on node classes — base-node__content has no explicit z-index so it may stack above the handle wrappers (z-index: 2). Handles are rendered by BaseNode.tsx via NODE_PORT_REGISTRY and positioned 6px outside the node bounds; no overflow clipping is set. Fix candidates: (1) add z-index: 0 to .base-node__content, (2) verify ReactFlow stylesheet is loading, (3) check for overflow:hidden on a parent wrapper.
